## When to include links in email

| Stage | Links OK? |
|-------|-----------|
| Cold first touch | NO (deliverability) |
| Follow-up after open | Yes, calendar |
| Warm inbound / referral | Yes, calendar |
| Trial / access request | Yes, platform signup + Notion guides |

## Anti-patterns

- Never send a Calendly link. Always use the HubSpot meetings URL above.
- Don't include links in a cold first-touch email. Ask for their availability instead.
