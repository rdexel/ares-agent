## Campaign file shape

Each file has:

1. **Hypothesis** — what we're testing
2. **Targeting** — list, filter criteria, ICP cuts
3. **Mechanics** — sequence, cadence, touch count
4. **What we measure** — the KPI(s) this campaign moves
5. **Current performance** — last snapshot + date
6. **Lessons so far** — what we've already learned (feeds playbook layer when it closes)
7. **Status** — live / paused / archived

Keep each file tight. If current performance is stale (>14 days old), flag it — either refresh the snapshot or flag that the campaign is no longer being watched.
